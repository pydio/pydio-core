Java-based uploader : not active by default, as Jumploader is not distributed with the application, Java applet to perform huge files uploads with progress, chunks files, and more. See http://jumploader.com/ and see below for installation instructions.

  • Drop the jumploader_z.jar inside the ajaxplorer plugins/uploader.jumploader/ folder
  • activate the plugin through the GUI (Settings > Global Configurations > Plugins > Uploader) by disabling the two other uploader plugins, and activating this one.
  • Go back to a file system repository, and clicking upload should open the Java applet!
